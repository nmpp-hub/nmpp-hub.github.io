GVEC is an open-source software for the generation of three-dimensional ideal magnetohydrodynamic (MHD) equilibria.

- Documentation: https://gvec.readthedocs.io
- Repository: https://gitlab.mpcdf.mpg.de/gvec-group/gvec
- Public Mirror: https://github.com/gvec-group/gvec
- Python Package: https://pypi.org/project/gvec
- Cite: Hindenlang et al., (2026). GVEC: A flexible 3D MHD equilibrium solver. Journal of Open Source Software, 11(120), 9670, https://doi.org/10.21105/joss.09670
